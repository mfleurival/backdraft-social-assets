# Backdraft Social Assets

Public image repository for Backdraft AI social media posts. Used by Paperclip AI agents to attach images to Buffer-scheduled posts.

## Structure

```
images/
  mockups/       — Screenshots of app mockup animations
  screenshots/   — Full-page and section captures of backdraftai.com
  branded/       — Composed images with branded backgrounds
```

## Usage

Images are referenced via raw GitHub URLs:
```
https://raw.githubusercontent.com/mfleurival/backdraft-social-assets/main/images/mockups/filename.png
```
